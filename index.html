    🔥 Synchronisation Firebase

    Pour synchroniser le tournoi en temps réel sur tous les appareils, configurez Firebase :

    1. Allez sur console.firebase.google.com
    2. Créez un projet → Activez Realtime Database (mode test)
    3. Dans les paramètres du projet → Config Web → copiez les valeurs
    4. Collez-les dans le fichier HTML à l'endroit indiqué firebaseConfig
    5. Remettez le fichier à jour sur GitHub

    ℹ️ Comment ça marche ?

    À chaque manche, les équipes s'affrontent selon leur classement :
    1er vs 2e · 3e vs 4e · 5e vs 6e · …

    Après saisie des scores → valider la manche → classement mis à jour → manche suivante.
    ⚠️ Le nombre d'équipes doit être pair.
